# 🚀 ProjectForge AI

AI-powered platform that generates **unique AI project ideas** using **Domain Fusion + Skill Intelligence**.

The system helps students and developers discover realistic projects based on:

- Selected technology domains
- Project difficulty
- Available development time
- Current skills
- Skills they want to learn

---

## 🌐 Live Demo

https://projectforgeai.streamlit.app

---

## ✨ Features

• AI-generated unique project ideas  
• Domain Fusion (combine multiple tech domains)  
• Skill-aware project generation  
• Project evaluation scoring  
• Auto PDF export for project ideas  
• Idea history tracking  

---

## 🧠 Example Use Case

A student selects:

```
Artificial Intelligence
Computer Vision
Cloud Computing
```

with skills:

```
Python, Machine Learning
```

and learning goal:

```
Docker
```

ProjectForge AI generates a **complete structured project idea**, including:

- Problem Statement
- System Architecture
- Tech Stack
- Dataset Suggestions
- Development Plan
- Project Evaluation Score
